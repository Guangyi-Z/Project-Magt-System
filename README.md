项目开发支持管理系统
=====
本系统是针对**项目开发人员&项目经理**而设计的项目支持管理系统，提供了必须的任务、成员管理功能，并提供相关的好友功能，方便用户管理相关领域的人际关系。

主要功能
-----

* 用户注册/登陆与管理员
* 项目任务管理
	* 任务的增删改查
	* 摘要任务与子任务
	* 任务先后关系链接
	* 自动生成甘特图
* 项目成员管理
	* 查看成员资料
	* 添加/删除成员
	* 设置成员的项目角色
* 项目好友管理
	* 添加/删除好友
	* 查找用户
	* 邀请好友加入项目
	* 查看好友资料
	* 修改备注名
* 用户个人信息管理
	* 查看个人信息
	* 修改密码

系统框架配置
-----
后端基于**JAVA的SSM框架**（STRUCT2, SPRING, MYBATIS）开发，前端主要采用**EXT JS和HTML,CSS技术**，数据库采用SQL SERVER 2008 R2，项目搭建在TOMCAT中间件上完成测试
